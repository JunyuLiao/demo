Part II

Interactive Algorithm with Design Mechanism 2

Study Overview

This is the second part of the user study. In this part users will interact with a similar algorithm, with different design mechanism. Specifically, users are required to explicitly indicate a few attributes you care about in prior. Users will then interact with the algorithm, focusing on the selected attributes.

How to Participate

You will first select 2-7 attributes that you care about most when choosing a car. Then you will interact with the algorithm using only these selected attributes to find your preferred options.

Click "Continue to Part II" to begin the attribute selection process.
