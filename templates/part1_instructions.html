Part I

Interactive Algorithm with Design Mechanism 1

Study Overview

This is the first part of the user study. In this part, users could interact with an algorithm and identify their favorite option among a car database. The database contains more than 5000 models, each described by 57 attributes. Typically it is hard for users to consider all attributes at the same time. Therefore, this system shows only a small subset of attributes each time to ensure user friendliness.

How to Participate

Within each question, two models along with a "not interested" option will be provided. Users are expected to selected an option based on their preference. If none of the attributes make sense, simply choose "not interested" to skip the question.
